প্রো-গ্রেড • রেসপন্সিভ • অফলাইন

CSS—শুরু থেকে আধুনিক পর্যন্ত, এক পাতায়।

নোট: CSS-এর “সকল” ফিচার এক পাতায় দেওয়া বাস্তবে অসম্ভব—তবে এখানে সর্বাধিক গুরুত্বপূর্ণ/আধুনিক টপিকগুলো সম্পূর্ণ ব্যাখ্যা ও ডেমোসহ রাখা হয়েছে, আর আপনি সহজেই বাড়াতে পারবেন।

আপনি কী শিখবেন?

  • ক্যাসকেড, স্পেসিফিসিটি, ইনহেরিট্যান্স
  • বক্স‑মডেল, ভিজ্যুয়াল স্টাইলিং
  • লেআউট: Flexbox, Grid, Position
  • রেসপন্সিভ: মিডিয়া ও কন্টেইনার কুয়েরি
  • অ্যানিমেশন, ফিল্টার, শেপ, মাস্ক
  • ভেরিয়েবল, ফাংশন, @rules (@media, @supports, @layer, @property)

ব্যবহার কিভাবে?

বাম পাশের মেনু থেকে সেকশন বাছুন। প্রতিটি সেকশনে ডেমো + কোড আছে। সব ডিভাইসে স্মুথ—মোবাইলে সাইডবার স্ক্রল হবে, কন্টেন্ট ফুল-উইডথ।

শর্টকাট: / চাপুন সার্চ খুলতে, Alt + ↑/↓ টিওসি নেভিগেট করতে।

ক্যাসকেড ও স্পেসিফিসিটি

CSS নির্ধারণ করে কোন স্টাইল জিতবে: উৎস (ব্রাউজার/ইউজার/অথর), ইম্পর্ট্যান্স (!important), স্পেসিফিসিটি (inline > id > class > element), এবং সোর্স অর্ডার।

উদাহরণ: স্পেসিফিসিটি 
/* কম > বেশি স্পেসিফিসিটি */

button { color: black; } .button.primary { color: blue; } #cta.button.primary { color: red; } /* ফলাফল: লাল */

@layer দিয়ে ক্যাসকেড কন্ট্রোল

@layer base, components, utilities; @layer base { button { font: inherit; } } @layer components { .btn { padding: .6rem 1rem; } } @layer utilities { .text-primary { color: dodgerblue; } }

সিলেক্টর ও পসুডো

বেসিক

  • টাইপ: div, ক্লাস: .card, আইডি: #box
  • অ্যাট্রিবিউট: input[type="email"]
  • কম্বিনেটর: A B, A > B, A + B, A ~ B

পসুডো‑ক্লাস

:hover, :focus-visible, :active, :disabled, :nth-child(), :not(), :is(), :where(), :has()

পসুডো‑এলিমেন্ট

::before, ::after, ::marker, ::selection, ::file-selector-button

  • CSS
  • Grid
  • Flex
কোড দেখুন 
/* :has() দিয়ে প্যারেন্ট স্টাইল */

.list:has(li.selected) { outline: 2px dashed var(--accent); }

ইউনিট, কালার, টাইপোগ্রাফি

ইউনিট

অ্যাবসোলিউট: px, cm, in • রিলেটিভ: em, rem, % • ভিউপোর্ট: vw, vh, svh, dvh • ফ্লুইড: clamp()

h1 { font-size: clamp(1.6rem, 2.5vw, 2.4rem); }

কালার

rgb(), hsl(), আধুনিক: oklch(), oklab() (ব্রাউজার সাপোর্ট দেখুন)।

button { color: oklch(55% 0.18 250); }

টাইপোগ্রাফি

লাইন‑হাইট, লেটার‑স্পেসিং, font-variation-settings, hyphens, text-wrap: balance;

p { text-wrap: pretty; hyphens: auto; }

বক্স‑মডেল ও ভিজ্যুয়াল

কনটেন্ট → প্যাডিং → বর্ডার → মার্জিন। box-sizing: border-box; দিয়ে সাইজিং সহজ।

.card { box-shadow: 0 10px 30px rgba(0,0,0,.25); border-radius: 14px; }

বর্ডার/আউটলাইন/শ্যাডো, ব্যাকগ্রাউন্ড‑ইমেজ, গ্রেডিয়েন্ট, background-attachment, clip-path।

.sticky { position: sticky; top: 8px; }

.box:hover { transform: translateY(-6px) scale(1.03); }
@keyframes কোড 
@keyframes spin { to { transform: rotate(1turn);} } .loader { animation: spin 1s linear infinite; }

ফিল্টার, ব্লেন্ড‑মোড, মাস্ক

কোড 
img { filter: contrast(1.1) saturate(1.1); }

img:hover { filter: blur(1px) contrast(1.2); } /* mix-blend-mode, mask-image ও backdrop-filter ও ব্যবহার করতে পারেন */

এডভান্সড: ভেরিয়েবল, ফাংশন, @rules

Custom Properties

:root { --gap: 12px; --brand: #6ea8fe; }

.box { gap: var(--gap); background: var(--brand); }

@property দিয়ে টাইপড কাস্টম প্রপার্টি অ্যানিমেটেবল:

@property --h { syntax: ''; initial-value: 0; inherits: false; } .box { background: oklch(70% .15 var(--h)); animation: hue 5s infinite linear; } @keyframes hue { to { --h: 360; } }

ফাংশন

calc(), min(), max(), clamp(), sin()/(experimental in some engines), conic-gradient() ইত্যাদি।

.fluid { width: clamp(280px, 60vw, 900px); }

@supports, nesting

@supports selector(:has()) { / modern enhancements / } / CSS Nesting (supported in modern browsers) */ .card { padding: 1rem; & a { color: dodgerblue; } & a:hover { color: deeppink; } }
